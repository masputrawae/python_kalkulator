## Panduan Cepat

### 1. Clone Repositori Ini

```bash
git clone git@github.com:masputrawae/python_kalkulator.git
```

### 2. Masuk ke Direktori

```bash
cd kalkulator
```

### 3. Jalankan Program

```bash
python3 main.py
```
